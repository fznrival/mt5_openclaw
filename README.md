# opnclaw_automated-trade

# MT5 + OpenClaw Trading Reporter
**Automated trading summary → Telegram via OpenClaw di Linux**

---

## 📁 Struktur File

```
mt5_openclaw/
├── mql5/
│   └── TradeExporter.mq5          ← EA di MT5, export trade ke CSV
├── python/
│   ├── trade_summary.py           ← Script utama: baca CSV → kirim Telegram
│   └── generate_sample_data.py    ← Buat data dummy untuk testing
├── openclaw_skill/
│   └── trading-reporter.skill.json ← Skill OpenClaw (trigger via Telegram)
├── setup_mt5_wine.sh              ← Install Wine + MT5 di Linux
├── setup_cron.sh                  ← Setup jadwal otomatis
├── trade_config.example.json      ← Template konfigurasi
└── README.md                      ← Ini
```

## 🚀 Setup Step by Step

### Step 1 — Install Wine + MT5

```bash
chmod +x ~/mt5_openclaw/setup_mt5_wine.sh
bash ~/mt5_openclaw/setup_mt5_wine.sh
```

Setelah selesai, buka MT5:
```bash
mt5
```

### Step 2 — Install EA di MT5

1. Buka MT5
2. Buka **File → Open Data Folder**
3. Masuk ke `MQL5/Experts/`
4. Copy file `TradeExporter.mq5` ke sana
5. Di MT5: **Navigator → Expert Advisors → klik kanan → Refresh**
6. Drag EA ke chart mana saja (misal XAUUSD H1)
7. Centang **Allow DLL imports** dan **Allow live trading**
8. Klik OK

EA akan otomatis export trade ke:
```
MT5_Files_Folder/trade_log.csv
```

### Step 3 — Buat Telegram Bot

1. Buka Telegram → cari **@BotFather**
2. Kirim `/newbot`
3. Ikuti instruksi, dapatkan **Bot Token**
4. Dapatkan Chat ID kamu:
   - Kirim pesan ke bot kamu
   - Buka: `https://api.telegram.org/bot<TOKEN>/getUpdates`
   - Lihat field `chat.id`

### Step 4 — Konfigurasi

```bash
# Buat file config
python3 ~/mt5_openclaw/python/trade_summary.py --setup

# Edit config
nano ~/.openclaw/trade_config.json
```

Isi config:
```json
{
  "telegram_bot_token": "123456:ABC-DEF...",
  "telegram_chat_id": "782858651",
  "csv_path": "/home/fznrival/mt5_data/trade_log.csv",
  "account_balance": 1000.0,
  "currency": "USD",
  "risk_per_trade_pct": 1.0,
  "timezone_label": "WIB"
}
```

> **Catatan:** `chat_id` kamu sudah ketahuan dari pairing OpenClaw tadi: **782858651**

### Step 5 — Test

```bash
# Buat data dummy dulu (tanpa MT5)
python3 ~/mt5_openclaw/python/generate_sample_data.py

# Test print tanpa kirim Telegram
python3 ~/mt5_openclaw/python/trade_summary.py --period week --dry-run

# Test kirim ke Telegram
python3 ~/mt5_openclaw/python/trade_summary.py --period week
```

### Step 6 — Setup Auto Schedule

```bash
chmod +x ~/mt5_openclaw/setup_cron.sh
bash ~/mt5_openclaw/setup_cron.sh
```

Jadwal otomatis:
- **Setiap Senin–Jumat jam 23:00 WIB** → Daily Report
- **Setiap Jumat jam 23:30 WIB** → Weekly Report
- **Akhir bulan jam 23:45 WIB** → Monthly Report

### Step 7 — Register OpenClaw Skill

```bash
# Salin skill ke folder OpenClaw
cp ~/mt5_openclaw/openclaw_skill/trading-reporter.skill.json ~/.openclaw/skills/

# Restart OpenClaw
openclaw gateway restart
```

Setelah ini, kamu bisa kirim perintah via Telegram ke OpenClaw:

```
"kirim summary trading hari ini"
"laporan mingguan trading"
"rekap profit loss bulan ini"
```

## 🔧 Troubleshooting

### CSV tidak ditemukan
```bash
# Cek path MT5 Files
find ~/.wine_mt5 -name "*.csv" 2>/dev/null

# Buat symlink manual
ln -sf "~/.wine_mt5/drive_c/Program Files/MetaTrader 5/MQL5/Files" ~/mt5_data/mt5_files
```

### MT5 tidak bisa dibuka
```bash
# Cek Wine
wine --version

# Cari executable MT5
find ~/.wine_mt5 -name "terminal64.exe" 2>/dev/null

# Jalankan manual
WINEPREFIX=~/.wine_mt5 wine "~/.wine_mt5/drive_c/Program Files/MetaTrader 5/terminal64.exe"
```

### Telegram tidak terkirim
```bash
# Test token langsung
curl "https://api.telegram.org/bot<TOKEN>/getMe"

# Cek log
tail -f /tmp/trade_summary.log
```

## ⚠️ Catatan Penting

- MT5 harus **tetap berjalan** agar EA bisa export CSV secara real-time
- Kalau laptop direstart, jalankan `mt5` dulu sebelum trading
- Chat ID kamu: **782858651** (dari pairing OpenClaw)
- File CSV di-overwrite setiap kali EA export (bukan append)
